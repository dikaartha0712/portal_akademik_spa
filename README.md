# Portal Akademik Mahasiswa (SPA) - Vanilla JS

## Cara menjalankan (paling mudah)
1. Buka folder ini di VS Code
2. Install extension **Live Server**
3. Klik kanan `index.html` -> **Open with Live Server**

## Catatan
SPA menggunakan History API. Jika Anda refresh di URL selain `/` pada server statis tanpa fallback, bisa 404.
Untuk tugas kelas biasanya aman via Live Server.
